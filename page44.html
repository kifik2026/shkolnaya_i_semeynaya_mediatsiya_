ИТОГОВЫЙ ТЕСТ по темам 12–15: «Семейная медиация»

 

Часть 1. Тест с выбором ответа

1. Семейная медиация — это:
а) судебное разбирательство
б) способ урегулирования конфликтов с участием посредника
в) психологическая диагностика
г) административная процедура

 

2. Основной принцип медиации:
а) принуждение
б) добровольность
в) конкуренция
г) контроль

 

3. Семья как система означает:
а) независимость членов семьи
б) взаимосвязь всех элементов
в) изоляцию участников
г) случайное взаимодействие

 

4. Гомеостаз семьи — это:
а) разрушение системы
б) стремление к стабильности
в) конфликт
г) развитие личности

 

5. Триангуляция — это:
а) сотрудничество
б) вовлечение третьего лица в конфликт
в) компромисс
г) избегание

 

6. Дифференциация «Я» означает:
а) зависимость
б) автономию при сохранении связей
в) изоляцию
г) агрессию

 

7. Генограмма используется для:
а) заключения договора
б) анализа семейной системы
в) оценки эмоций
г) контроля поведения

 

8. Рефрейминг — это:
а) давление
б) смена темы
в) изменение интерпретации
г) отказ от общения

 

9. Активное слушание включает:
а) игнорирование
б) перебивание
в) перефразирование
г) оценку

 

10. Основная задача медиатора:
а) принять решение
б) помочь сторонам договориться
в) наказать виновного
г) контролировать процесс

 

Часть 2. Углубленные вопросы

11. Коалиции в семье — это:
а) случайные связи
б) устойчивые союзы между членами семьи
в) конфликты
г) правила поведения

 

12. Эмоциональный разрыв проявляется в:
а) усилении контакта
б) дистанцировании
в) сотрудничестве
г) компромиссе

 

13. Семейная проекция — это:
а) передача имущества
б) перенос эмоций родителей на ребенка
в) воспитание
г) обучение

 

14. Основной этап медиации:
а) наказание
б) выявление интересов
в) контроль
г) завершение

 

15. Коммуникативные дефициты — это:
а) навыки общения
б) трудности взаимодействия
в) лидерство
г) мотивация

 

16. Основная цель семейной медиации:
а) победа стороны
б) восстановление взаимодействия
в) наказание
г) разрыв отношений

 

17. Системный подход предполагает:
а) анализ личности
б) анализ связей
в) анализ эмоций
г) анализ поведения

 

18. Балансировка — это:
а) контроль
б) обеспечение равенства сторон
в) давление
г) игнорирование

 

19. Эмоциональная регуляция направлена на:
а) усиление конфликта
б) снижение напряжения
в) прекращение общения
г) контроль

 

20. Принцип конфиденциальности означает:
а) открытость информации
б) неразглашение
в) обсуждение
г) контроль

 

Часть 3. Соответствие

Понятие

Характеристика

2. Проекция

б) автономия

3. Треугольник

4. Разрыв

г) дистанция

 

22. Соотнесите этапы медиации:

Этап

Содержание

а) поиск решений

2. Выявление интересов

б) анализ ситуации

3. Генерация решений

в) потребности

4. Соглашение

г) фиксация

 

Часть 4. Ситуационные вопросы

23. Участник избегает общения. Это:
а) коалиция
б) разрыв
в) компромисс
г) диалог

 

24. Родитель вовлекает ребенка в конфликт — это:
а) баланс
б) триангуляция
в) сотрудничество
г) компромисс

 

25. Один участник доминирует:
а) равенство
б) дисбаланс
в) сотрудничество
г) компромисс

 

26. Участники обвиняют друг друга:
а) конструктив
б) позиционное мышление
в) сотрудничество
г) компромисс

28. Перечислите 3 трудности семейной медиации.
